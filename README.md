## Features

- Format Support

  - [x] LRC
  - [ ] KRC
  - [ ] QRC
  - [ ] YRC
  - [ ] TTML

- Export Lyric

  - [ ] LRC
  - [ ] KRC
  - [ ] QRC
  - [ ] YRC
  - [ ] TTML

- Parse Main Lyric

  - [x] Original
  - [x] Dynamic

- Parse Extended Lyric

  - [x] Translate
  - [x] Roman
  - [ ] Unknown

- Parse Meta

  - [x] Meta Tag
  - [x] Match Producer

- Extra

  - [x] Insert Space
  - [x] Insert Interlude Line
  - [x] Insert Duet Info
  - [x] Purification Lyrics

## Install

```shell
npm music-lyric-kit
```

## Usage

```js
import { LyricParser } from 'music-lyric-kit'

const parser = new LyricParser()

const origin = ''
const dynamic = ''
const translate = ''
const roman = ''

const result = parser.lrc.parse({ original, dynamic, translate, roman })

console.log(result)
```
